
## Notes
### Disjoint set(Array)
**Operations**  
init: 第 x 人在第 x 團  
union: 合併兩團     
find: 某個人在第幾團？  
equivalence: 兩人是否同團？  
singleton: 團體是否合併過？  
cardinality: 一個團裡有幾人？  


**Time complexity:**  
union: O(N)  
find: O(1)  
equivalence: O(1)  
cardinality: O(1)  
singleton : O(1)
